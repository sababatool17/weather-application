This is a simple Weather Application built using Python and Tkinter.  
The app allows users to enter the name of a city and check the current temperature using the OpenWeatherMap API.
## Features
- Simple graphical user interface (GUI)
- Search weather by city name
- Displays temperature in Celsius
- Shows whether the weather is hot or cold
- Uses real-time weather data from API
## Technologies Used
- Python
- Tkinter
- Requests Library
- OpenWeatherMap API
## Installation
1. Install Python
2. Install the required package:
pip install requests
3. Run the Python file:
python weather_application.py
## How It Works
1. Enter the name of a city
2. Click the "Find It!" button
3. The app fetches weather data from OpenWeatherMap
4. Temperature and weather condition are displayed
## Example
- Istanbul → 18°C → It is cold
- Dubai → 35°C → It is hot
## API Used
OpenWeatherMap API  
https://openweathermap.org/api
